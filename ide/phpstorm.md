# 快捷方式

-   Navigate | Line/Column   跳转到某行 `command + L` 
-   Find | Add Selection for Next Occurrence 选中当前单词可继续选中下一个单词,这个需要自己加一下 `command + D`

-   Code | Move Line Up , Move Line Down 上下移动当前行 `shift + option + up` / `shift + option + down`
